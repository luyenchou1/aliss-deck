# ALISS — Agentic memory for dynamic businesses

A short explainer deck on **ALISS** (Agentic Longitudinal Intelligent Storage System), the memory architecture behind DeweyLearn's business intelligence stack.

**View it:** [open the deck](https://luyenchou1.github.io/aliss-deck/) · [print version](https://luyenchou1.github.io/aliss-deck/print.html)

## What's in here

- `index.html` — the deck (9 slides, scroll to read)
- `print.html` — print-optimized variant
- `images/` — slide artwork

The deck renders client-side using React 18 + Babel via CDN. No build step — open `index.html` in any modern browser.
